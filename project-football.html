Field Goal Video Game with Unity

About the Game

The field goal video game was a project a friend and I worked on. The object is very simple: kick the football through the uprights to get points, with random distance in both the X and Y directions, as well as wind, making the task more challenging than it looks.

Languages & Software Used

Game Features

User controls the launch angle and power via their respective meters.

The ball starts at a random point within set parameters a certain distance away from the uprights, making every round different.

A wind of random strength and direction affects the ball in the air.

Messages are given to the user based on different hitbox interactions, such as making or missing a field goal or hitting the uprights.

Points are added to the user's score upon a made field goal, with farther attempts with strong wind worth more.
